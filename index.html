ABOUT

While working as a TV Producer in 2019 I was introduced to programming by a friend. After learning the basics of HTML and CSS I immediately started building simple pages. Soon after I began studying Javascript from a book and finally felt that I was doing something that I really loved.

This was different to what I was doing up to that point but what I found out was that it really suited my personality type. After solving hundreds of coding challenges it all started to make a lot more sense. It gave me the confidence and it taught me that web development needs to be approached as a set of problems that need to be figured out.

I haven’t stopped coding since and have created a portfolio of projects that expanded my knowledge and strengthened my passion even more.
